# OpenBoxOffice Landing Page

This repository holds the contents of the landing page for OpenBoxOffice located [here](https://openboxoffice.org). This app is built with Vite, React, and TailwindCSS for a modern Single Page App (SPA) experience. Feel free to contribute by opening a PR!
